Files:
- `vizior_mark.svg` / `vizior_mark.png`: ringed Vizior monogram
- `vizior_badge.svg` / `vizior_badge.png`: presentation badge
- `vizior_wordmark.svg` / `vizior_wordmark.png`: mark + wordmark lockup

- Accent red: `#DA303C`
- Bright accent: `#F5525C`
- White: `#F5F7FA`
